# Добавление нового проекта для кэширования

## Описание

Нужно добавить проект-библиотеку для предоставления удобного интерфейса сервисов кэширования
На текущий момент, нужно добавить технологию для кэширования Redis и кэш в памяти приложения -
`Microsoft.Extensions.Caching.Memory.MemoryCache`, но также, нужно оставить возможность для добавления
новой технологии или лёгкой замены текущей.

В рамках этой задачи нужно добавить интерфейсы для сервисов кэширования.

Предварительно, интерфейс сервиса кэширования будет такой

```csharp
public interface ICache
{
    Task<TValue> GetOrSetAsync<TValue>(string key, Func<string, Task<TValue>> factory);
    Task<TValue?> TryGetAsync<TValue>(string key);
    Task SetAsync<TValue>(string key, TValue value);
    Task RemoveAsync(string key);
    Task ExpireAsync(string key);
    Task ClearAsync();
}
```

Оговорюсь, что это лишь примерный облик контракта сервиса кэширования. Нужно подумать, как будет лучше
Возможно, некоторые методы нужно удалить, некоторые добавить, некоторые изменить

Кроме сервиса кэширования должен быть удобный интерфейс по конфигурации кэширования:

* Какую технологию использовать
* Какие политики кэширования использовать
* Другие настройки

Подозреваю, что для каждой технологии кэширования будет свой конфигуратор, который можно получить из единой точки входа.
