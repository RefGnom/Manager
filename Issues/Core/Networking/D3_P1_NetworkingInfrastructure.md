# Построение инфраструктуры сетевого взаимодействия

## Описание

Уже существует проект `Manager.Core.Networking`, который должен предоставлять сетевую инфраструктуру, нужно работать в
нём.

На данном этапе нужно подумать над разными решениями организации сетевой инфраструктуры, которая должна включать в себя
следующие возможности:

* Retries
* Fallback
* Rate limiting
* Authorization by api key

Сейчас в сетевом взаимодействии уже есть авторизация по апи ключу, её нужно встроить в общую инфраструктуру сетевого
взаимодействия
